## Flex布局


### flex:1
完整版是
```css
.item{
	flex-grow: 1; /* 空间多余时的分配“份数” */
	flex-shrink: 1; /* 空间不足时的分配“份数” */
	flex-basis: 0%;  /* 分配空间前所占的空间，可以是绝对单位也可以是相对单位（包含块） */
}
```
