## 数组扁平化
核心就是`Array.isArray()`、`depth`

```js
function flatten(arr, depth = 1) {
    const result = [];

    arr.reduce((flat, item) => {
        if (Array.isArray(item) && depth > 0) {
            flat.push(...flatten(item, depth - 1));
        } else {
            flat.push(item);
        }

        return flat;
    }, result)

    return result;
}
```
